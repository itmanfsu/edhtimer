# Changelog

All notable releases of Nerd Cave Comander Clock are documented here.

## [1.0.0] - 2026-07-18

First stable release.

### Included

- Configurable two-to-six-player Commander turn timer
- Optional per-player elimination clock
- Round-aware turn counter
- Firebase shared rooms with anonymous authentication
- Firebase App Check using reCAPTCHA Enterprise
- Realtime multi-device controls and timestamp-based synchronization
- Twelve-hour inactive-room expiration and manual room deletion
- Document Picture-in-Picture floating timer
- Translucent bookmarklet overlay
- Installable Progressive Web App for Android and iPhone
- Buildkite deployment to `edhtimer.com`
